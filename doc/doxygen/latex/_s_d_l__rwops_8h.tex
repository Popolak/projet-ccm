The data is allocated with a zero byte at the end (null terminated)

If {\ttfamily datasize} is not NULL, it is filled with the size of the data read.

If {\ttfamily freesrc} is non-\/zero, the stream will be closed after being read.

The data should be freed with SDL\+\_\+free().
